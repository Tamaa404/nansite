Today, I started working on Astro-Light—a static portfolio built with Astro and Tailwind CSS. The goal is to create a site that feels professional but loads instantly, even on slower connections.

## Why Astro?

Astro is perfect for this because:
- **Zero JavaScript** by default
- **Static generation** means no database needed
- **Content Collections** handle Markdown like a pro
- **Image optimization** is built-in

## The Architecture

The site is organized into four zones:
1. **Home** - Landing page with recent updates
2. **Lab** - Project showcase (AI, robotics, web dev)
3. **Classroom** - Teaching modules (Arduino & Python)
4. **Feed** - Life update chronological feed

Stay tuned as I build this out in public!
